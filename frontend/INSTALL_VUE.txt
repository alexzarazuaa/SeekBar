----------------------------------------------------------------- INSTALACION VUE ------------------------------------------------------------------

PASOS : 

   Instalar  Vue CLI 

         sudo npm install -g @vue/cli

    Ver la version de vue

        vue --version
    
    



 Para crear unn nuevo proyecto vue


        vue create hello-world  

        *  Seleciona la Version vue3 *
    
    Luego ya : 

        npm install
        npm run build
        npm run serve
